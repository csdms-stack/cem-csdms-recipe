# Installing csdms-cem

To install csdms-cem from the csdms-stack channel with `conda`:

```bash
$ conda config --add channels conda-forge
$ conda config --add channels csdms-stack
```

Once these channels have been activated:

```bash
$ conda install csdms-cem
```

It is possible to list all of the versions of csdms-cem available on your
platform with:

```bash
$ conda search csdms-cem --channel csdms-stack
```
